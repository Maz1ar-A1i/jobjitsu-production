# Agent Knowledge Base

This file contains patterns, conventions, and learnings for AI agents working on this codebase.
Always read this file before starting a task.

## Codebase Patterns
- **Frontend**: React (Vite/CRA), JavaScript/JSX.
- **Backend**: Node.js/Express (from context).
- **Review**: Always run typechecks and linting before committing.

## Workflow Rules (GSD-Ralph)
- **Atomic Commits**: One task = one commit.
- **Log Progress**: Append learnings to `progress.txt`.
- **Browser Verify**: Always verify UI changes in browser.
