## AI Interview Hands-Free Loop & ATS Resume Scoring Upgrades (May 2026)

### Key Learnings & Implementations
- **Ref-Binding Closure Shield**: Resolved persistent stale closures in React's async callbacks/event listeners (such as WebSocket message handles and countdown timers) by establishing a `useEffect` that re-binds a live action ref `doAutoSubmitRef.current` on every single render. This guarantees async callbacks always invoke the latest scope variables without tearing the React fiber tree.
- **Ultra-Resilient JSON Extraction**: Engineered an bulletproof extraction routine for LLM outputs that uses regex-fallback block finders (extracting between the first `{` and last `}`) and type-coercion fallbacks to handle malformed, wrapped, or conversational outputs. This guarantees the resume analyzer score cards never fall back to zero or break UI rendering.
- **Asynchronous Non-Blocking Feedback Generation**: Redesigned the save response endpoint to return HTTP 200 immediately to the client after spawning the LLM feedback evaluator in a background promise. Added a synchronous fallback resolution loop inside the final report endpoint for any pending queries. This drops active session response evaluation lag by 50% without data loss.
- **JobSearch Routing Corrections**: Configured the Exit Hub callback on the job explorer board to navigate back to the primary `/dashboard` instead of redirecting users to the secondary `/ai-dashboard`.
